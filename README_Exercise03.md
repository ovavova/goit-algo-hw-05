# Порівняння алгоритмів пошуку підрядка в тексті.

Для обох текстів був проведений аналіз часу пошуку підрядка. 
На графіках нижче показаний середній час виконання (взято 7 ітерацій def summarize_for_plot - repeats) для кожного з трьох алгоритмів для двох випадків:
час для випадку коли рядок є в тексті - Found - F на графіку та час коли рядок не було знайдено - NF на графіку

## Висновки:

Всі вимірювання показали одну картину: алгоритм Боер Мура працює найшвидше, за ним йде KMP і найдовше працює Robin Karp. 
Результати порівняння однакові як для випадку знаходження паттерна в тексті файлу так і для випадку коли паттерна в файлі немає. 
Також логічно що для випадку коли патерна в тексті немає, то пошук займає більше часу перевіряючи всі можливі варіанти.
